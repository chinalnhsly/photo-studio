《影楼商城》微信小程序开发计划

一、开发流程指引
（一）在微信开发者工具中预览
打开微信开发者工具。
选择 “项目 -> 导入项目”。
项目目录选择：/home/liyong/photostudio/mobile-app/dist。
若没有 AppID，可使用测试号。
勾选 “不校验合法域名、web - view（业务域名）、TLS 版本以及 HTTPS 证书”。
（二）项目优化建议
参考《项目优化清单.md+25 - 1》进行项目优化。
